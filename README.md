# Expense Tracker (Local-First)

A minimal, action-first expense tracker built as a static web app.

No backend.  
No authentication.  
No frameworks.  
No tracking.  

---

## 🎯 Philosophy

- Fast daily logging
- Monthly-first clarity
- Numeric analytics only
- Zero noise

Log fast. See clearly. Leave.

---

## 🚀 Features

- +Protein quick add
- Manual expense entry
- Decimal support
- Category breakdown
- Last 12 months overview
- Local-only storage (browser)

---

## 🌍 Deployment

Hosted using GitHub Pages.

Enable in:

Settings → Pages → Deploy from branch → main → / (root)

---

## 🔒 Privacy

All data is stored locally in your browser.

Clearing browser storage will delete your data.

No data is sent anywhere.
